# SAtraceで収集したデータの分析


# データ形式
* ファイル名
	* タイムスタンプ`%Y%m%d_%H%M%S.txt`
	* 一日5分間隔
	* .txt形式


* ファイルの内容
	* 4列、1001行+ヘッダーフッター
	* サンプルデータはヘッダーフッター省略
	* 1列目はインデックス1~1000
	* 2,3,4列目はガウシアン分布に従う適当な値
	* x,y軸の順に1,2列目、1,3列目、1,4列目のペアで描くとガウシアン描く
	* 空白詰め


```ファイルの内容
<ヘッダー>
1       3.72665317e-06   3.91752688e-06   4.11776507e-06 
2    ...............
3
4
.
.
.
1001.......................
<フッター>
```
