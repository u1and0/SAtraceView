使い方¶

shift+enterで「セルの実行」+「次のセルへ移動」

準備¶

  • グラフのインライン化

    %matplotlib.inline

  • モジュールのインポート

    import plottxt as pt

%matplotlib inline

  • import pandas as pd
  • import numpy as np

  • from scipy import stats
  • import glob
  • import json
  • from tqdm import tqdm

conda install tqdm

または

conda install -c conda-forge tqdm=4.10.0

をコンソールに打ち込んでインストール

plot()メソッドでグラフの表示。

In [5]:
pt.spectrum(path + file).plot()
Out[5]:
<matplotlib.axes._subplots.AxesSubplot at 0xb75b630>

parameter.jsonに格納されたfreq関連の値から適当にインデックスを割り振ってくれる。

  • freq_center: 中心周波数
  • freq_span: 周波数のスパン
  • number_of_rows: データ列は何行あるか。

parameter.jsonを開いてこれらの値をいじれば、スパンを変えた日付2016/10/28以降のデータもグラフかできます。

引数columnsにmax, minを打つと、txtファイルのMax値, Min値に切り替え可能。

(以降は使わないけど、オプションとしてあることを示しておきます。大文字小文字無視)

In [19]:
pt.spectrum(path + file, columns='max').plot()
pt.spectrum(path + file, columns='mIn').plot()
Out[19]:
<matplotlib.axes._subplots.AxesSubplot at 0xc63b7f0>

複数ファイルの処理¶

電界強度のグラフ化¶

グラフ化したい複数のファイルを ()の中にくくって複数指定する。

In [22]:
files = ('20151125_002404.txt',
        '20151214_225858.txt',
        '20161013_091345.txt')

pythonの内包表記使ってpathを足したフルパスのリストを作成し、 それをspectrum_many関数へ渡す。

In [23]:
df = pt.spectrum_many([path + i for i in files])
df.plot()
Out[23]:
<matplotlib.axes._subplots.AxesSubplot at 0xc16bc50>

ノイズフロア¶

ノイズフロアを知りたいときは、pandas.DataFrameまたはpandas.Seriesの後に.noisefloor()

In [15]:
df.noisefloor()
Out[15]:
2015-11-25 00:24:04    -80.20
2015-12-14 22:58:58    -98.26
2016-10-13 09:13:45   -106.72
dtype: float64

SN比¶

さきほどのdfからノイズフロアを引き算してやれば、SN比が出る。

In [18]:
sn = df - df.noisefloor()
sn.plot()
Out[18]:
<matplotlib.axes._subplots.AxesSubplot at 0xc086a90>

最大値だけプロット¶

SN比のデータ集合であるsnから最大の値だけを抜き出すとき、 max(axis=1)メソッドでそれぞれの周波数の最大値だけを抜き出す。 (axis=0はそれぞれの日時の最大値)

In [25]:
sn.max(axis=1).plot()
Out[25]:
<matplotlib.axes._subplots.AxesSubplot at 0xc9ca940>
